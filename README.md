> 這是一個 Anime1.me 的**非官方** app，使用 React Native 製作。

## 特色與功能
- 動畫列表
- 新番時間表
- 收藏動畫
- 觀看紀錄
- 適合動畫 App 的顯示介面
- 內建播放器
  - 可調整播放速度
  - 前進、後退十秒
